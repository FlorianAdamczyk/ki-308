# Ressourcen

## nbstripout

Ein Tool, um die Ausgabe von Jupyter Notebooks zu bereinigen, bevor sie in ein Versionskontrollsystem wie Git eingecheckt werden. Es entfernt die Ausgaben von Zellen, sodass nur der Code und die Markdown-Inhalte im Notebook verbleiben.

### Installation
z.B.:
```bash
pip install nbstripout
```

### Verwendung

```bash
nbstripout --install
```

Dies installiert nbstripout als Git-Hook, sodass es automatisch ausgeführt wird, wenn ihr Änderungen an einem Jupyter Notebook vornehmt und diese in Git eincheckt.
